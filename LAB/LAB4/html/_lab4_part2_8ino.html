Detailed Description

Outputs a current and averaged lightlevel to an LCD screen, plays a sound if the light level is outside a threshold.

This file contains the code for a system that reads the light level from a photoresistor and outputs the current light level and a simple moving average to an LCD screen. If the light level is outside a threshold, a sound is played on a buzzer. Additionally, the system calculates and prints prime numbers from 2 to 5000.
