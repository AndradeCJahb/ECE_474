Detailed Description

Final Project for EECSE 474 Embedded Systems including a couple game tasks to be completed to win or "Debug" the system.

The project includes a Joystick minigame and a RTC task to be completed to win the game. The game also includes a displayed timer and buzzer to indicate time remaining.

◆ BuzzerTimeTask()

void BuzzerTimeTask ( void * pvParameters)

Task to control buzzer frequency based on time remaining.

Uses the ledc library to blink buzzer on and off at decreasing intervals as the time gets lower and lower. Intervals are 2 seconds, 1 second, 0.5 seconds, 0.25 seconds

◆ GameEndTask()

void GameEndTask ( void * pvParameters)

Task to handle game timer, including win and lose conditions.

Detects win and lose conditions with boolean flag variables and a global countdown timer, displaying the current state of the game on the LCD

◆ JoystickGameTask()

void JoystickGameTask ( void * pvParameters)

Task to handle joystick game sequence and user input.

Takes advantage of the freeRTOS queues, receiving values from both to construct a list of directions and a sequence of inputs to compare them to. Fails the game if the user provides incorrect inputs.

◆ JoystickInputTask()

void JoystickInputTask ( void * pvParameters)

Task to read joystick input and send to queue.

Reads analog voltage input from a joystick peripheral, and sends it to the AnalogX queue. Contains a debouncing mechanism to prevent unintentional inputs.

◆ JoystickProducerTask()

void JoystickProducerTask ( void * pvParameters)

Task to generate random sequence of left and right directions and send to queue.

This task uses C's rand() function to generate binary values (either 1 or 0) and sends them to the freeRTOS queue before deleting itself
